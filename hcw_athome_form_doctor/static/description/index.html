HCW@Home - remote consultation integration

HCW@Home is an Open Source Teleconsultation system, with video call support. This addon does nothing else than adding configuration options into Odoo Settings. This module is a dependancy of hcw_athome_form or hcw_athome_form_doctor, please read the according module documentation for more information. If you will need your own HCW@Home instance to use this addon.

HCW@Home is maintained by Iabsis SARL based in Switzerland.

Configuration

  • Open Odoo Settings and define the various HCW@Home URL and informations.
  • Install one of the module hcw_athome_form or hcw_athome_form_doctor
